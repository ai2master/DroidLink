# DroidLink -- 安全优先的 Android 手机管理工具

## 一句话定位

通过 USB 数据线管理 Android 手机的桌面工具，所有数据传输不经过任何网络。

---

## 核心功能

### 屏幕镜像与远程操控
在桌面上实时显示手机屏幕，用鼠标和键盘操控手机。支持自定义分辨率、比特率和帧率。

### 数据同步
自动同步联系人、短信和通话记录到桌面端，支持搜索、筛选和浏览。数据变更基于哈希检测，只同步有变化的内容。

### 版本历史与数据恢复
每次同步自动创建版本快照。支持：
- 丰富的数据预览（非原始 JSON）
- 任意两个版本之间的字段级对比（高亮新增、删除、修改）
- 一键恢复到任何历史版本（创建新版本，不破坏历史记录）

### 剪贴板共享
桌面和手机之间双向共享剪贴板。完整支持中文和所有 Unicode 字符。

### 文件管理
浏览手机文件系统，上传和下载文件。支持设置文件夹同步规则，在指定目录之间自动增量同步。

### 中文/CJK 输入
内置 DroidLinkIME 输入法，从桌面键盘直接向手机输入中文、日文、韩文。解决了 scrcpy 无法输入非 ASCII 字符的痛点。

### 多语言界面
桌面端和 Android 端均支持 7 种语言：简体中文、English、日本語、한국어、Русский、Deutsch、العربية。

### Companion App 自动安装
手机插入电脑后自动检测并提示安装 Companion App。需要用户明确同意后才会安装，不会静默安装。

## 安全特性

这是 DroidLink 与所有竞品的根本区别。

### 零网络通信
- 桌面端与 Android 端之间 **不建立任何网络连接**
- 不使用 Wi-Fi、蓝牙、TCP、HTTP、WebSocket
- 不使用 `adb forward` 端口转发
- 所有数据通过 USB 数据线上的 ADB 协议传输

### Android 端无网络能力
- Companion App **不申请 INTERNET 权限**
- 即使设备被 Root，Companion App 也无法建立网络连接
- 不依赖任何 HTTP 服务器库

### 被动模式
- Android 端是完全被动的，只响应桌面端发来的 ADB 命令
- Android 端不会主动发起任何连接
- 物理断开 USB 线即可立即切断所有通信

### 文件系统安全
- 桌面端访问范围受 Tauri 安全策略限制
- 明确禁止访问 SSH 密钥、GPG 密钥和证书文件
- CSP（Content Security Policy）限制 WebView 的加载来源

### 透明的数据流
- 所有数据传输都通过 ADB 命令，可通过 `adb logcat` 完整审计
- 不存在后台静默传输
- 用户可以随时通过拔掉 USB 线停止一切通信

## 使用场景

### 开发者日常
- 在桌面上操控手机进行测试和调试
- 从桌面键盘输入中文到手机应用
- 快速在桌面和手机之间传输测试文件

### 数据备份
- 定期同步联系人、短信、通话记录到桌面端
- 版本历史保留完整的变更记录
- 误操作后可恢复到任何历史版本

### 企业/安全敏感场景
- 管理不允许联网的内部设备
- 在隔离网络环境中管理手机
- 需要确保数据不经过第三方服务器的场景

### 办公辅助
- 在电脑上查看和回复手机短信
- 从电脑剪贴板粘贴内容到手机
- 在大屏幕上操作手机应用

### DroidLink 的独特优势

1. **安全性无可比拟**：唯一一个 Android 端完全不需要 INTERNET 权限的管理工具。数据传输链路可审计，不存在中间人攻击的可能。

2. **版本历史**：唯一提供联系人/短信/通话记录版本历史、对比和恢复功能的工具。

3. **CJK 输入**：通过 DroidLinkIME 解决了 scrcpy 系工具无法输入中文的问题，且不需要网络输入法。

4. **物理隔离**：拔掉 USB 线 = 完全断开连接。不存在后台残留连接。

## 技术亮点

- **Tauri v2 架构**：Rust 后端保证性能和内存安全，React 前端保证界面体验
- **增量同步**：基于 xxh3 哈希的变更检测，只传输有变化的数据
- **不可变版本历史**：恢复操作创建新版本记录，历史记录永远不会被修改或删除
- **7 语言国际化**：桌面端和 Android 端均完整国际化
- **自动 ADB 管理**：内置 ADB，自动检测并避免与系统已安装的 ADB 冲突

- Android 8.0 (Oreo) 及以上版本
- 覆盖约 95% 的 Android 设备
