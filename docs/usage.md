# DroidLink 使用说明

## 系统要求

### 桌面端

| 平台 | 最低版本 |
|------|---------|
| Windows | 10 (64-bit) |
| macOS | 10.15 Catalina |
| Linux | Ubuntu 20.04 / Fedora 33 / Arch (64-bit) |

### Android 端

- Android 8.0 (Oreo, API 26) 或更高版本
- 已开启 USB 调试模式

### 硬件

- USB 数据线（非纯充电线）
- 至少 200 MB 可用磁盘空间

---

## 安装

### 桌面端安装

#### Windows
1. 下载 `DroidLink_x.x.x_x64-setup.exe`
2. 双击运行安装程序
3. 按提示完成安装

#### macOS
1. 下载 `DroidLink_x.x.x_x64.dmg`
2. 打开 DMG 文件，将 DroidLink 拖入 Applications 文件夹
3. 首次运行时右键选择"打开"以绕过 Gatekeeper

#### Linux
```bash
# Debian/Ubuntu (.deb)
sudo dpkg -i droidlink_x.x.x_amd64.deb

# 或使用 AppImage
chmod +x DroidLink_x.x.x_amd64.AppImage
./DroidLink_x.x.x_amd64.AppImage
```

### Android 端安装

**无需手动安装。** 当你通过 USB 连接手机到电脑时，DroidLink 会自动检测是否已安装 Companion App，并弹出安装提示。

安装流程：
1. 用 USB 线连接手机和电脑
2. DroidLink 检测到手机后弹出"安装 Companion App"对话框
3. 对话框会说明 Companion App 的作用（数据同步、剪贴板、中文输入等）
4. 点击 **"安装 Companion App"** 按钮开始安装
5. 手机上可能弹出安装确认提示，点击"允许"
6. 安装完成后在手机上打开 DroidLink Companion，授予所有权限

你也可以选择 **"暂时跳过"** 跳过安装，但部分功能（联系人同步、短信同步等）将不可用。

## 首次连接

### 1. 开启 USB 调试

在 Android 手机上：
1. 打开 **设置** > **关于手机**
2. 连续点击 **版本号** 7 次，开启开发者模式
3. 返回 **设置** > **开发者选项**
4. 开启 **USB 调试**

### 2. 连接手机

1. 用 USB 数据线连接手机和电脑
2. 手机上弹出"允许 USB 调试"对话框，勾选"始终允许"，点击 **确定**
3. DroidLink 自动检测到设备，状态栏显示"已连接"

### 3. 授予 Companion App 权限

首次启动 Companion App 时，需要授予以下权限：
- 联系人读取/写入
- 短信读取
- 通话记录读取
- 手机状态
- 通知权限（Android 13+）

## 功能说明

### 仪表盘

连接设备后的主页面，显示：
- 设备信息（型号、Android 版本、电池电量）
- 存储空间使用情况
- 联系人/短信/通话记录数量统计
- 同步状态概览
- 一键触发同步

### 屏幕镜像

实时显示手机屏幕画面，支持从桌面端操控手机。

**操作方式：**
- 鼠标左键 = 触屏点击
- 鼠标拖动 = 触屏滑动
- 鼠标滚轮 = 上下滚动
- 键盘输入 = 英文文本输入

**设置项：**
- 分辨率：最大分辨率可调（降低分辨率可减少延迟）
- 比特率：视频质量，默认 8 Mbps
- 帧率：默认 60 fps
- 转发所有点击：包括中键和右键
- 保持唤醒：镜像时防止手机锁屏

### 联系人管理

- 查看手机上所有联系人（姓名、电话、邮箱、组织）
- 搜索和筛选
- 展开行查看版本历史，包括：
  - 每次变更的时间和操作类型
  - 点击眼睛图标查看版本详情（丰富预览，非原始 JSON）
  - 点击对比图标与上一版本进行字段级对比（高亮显示新增/删除/修改）
  - 点击恢复图标将数据恢复到该版本（创建新版本记录，不破坏历史）

### 短信管理

- 查看所有短信（收件箱、已发送、草稿）
- 按联系人分组，支持搜索
- 点击"版本历史"按钮查看短信同步历史
- 每条版本记录可预览和恢复

### 通话记录

- 查看所有通话记录（来电、去电、未接）
- 显示通话时长、联系人名称
- 点击"版本历史"按钮查看变更历史
- 支持版本预览和恢复

### 剪贴板共享

在桌面端和手机端之间共享剪贴板内容：
- 桌面复制的文本可粘贴到手机
- 手机复制的文本可粘贴到桌面
- 完整支持中文和 Unicode 字符

### 文件管理器

浏览和管理手机上的文件系统：
- 目录浏览和导航
- 文件上传（桌面 -> 手机）
- 文件下载（手机 -> 桌面）
- 文件删除

### 文件夹同步

在桌面和手机之间设置文件夹同步规则：
- 添加同步文件夹对（桌面路径 <-> 手机路径）
- 支持同步方向：双向 / 仅上传 / 仅下载
- 文件变化检测和增量同步
- 同步状态实时显示

### 中文/CJK 输入（DroidLinkIME）

通过桌面键盘向手机输入中文、日文、韩文等 CJK 文字：

**使用前提：**
1. Companion App 已安装
2. 在手机上启用 DroidLink 输入法：设置 > 系统 > 语言和输入法 > 管理键盘 > 开启 DroidLinkIME
3. 在需要输入时切换到 DroidLink 输入法

**使用方式：**
- 在屏幕镜像界面，直接用桌面键盘输入中文
- 输入内容通过 ADB 广播发送到手机端的 DroidLinkIME 服务
- 也可以随时切换回手机原生输入法

### 版本历史

独立的版本历史管理页面：
- 按数据类型筛选（联系人/短信/通话记录）
- 时间线展示所有版本变更
- 对比模式：勾选两个版本进行并排对比
- 恢复操作：选择任一版本恢复为当前状态（创建新版本，不删除旧版本）

### 设置

- **语言切换**：支持 7 种语言
  - 简体中文
  - English
  - 日本語
  - 한국어
  - Русский
  - Deutsch
  - العربية
- **同步间隔设置**
- **ADB 路径配置**
- **输入法模式切换**（DroidLinkIME / 手机原生输入法）

## 安全说明

DroidLink 的所有数据传输通过 USB 数据线完成，不经过任何网络：

- 不使用 Wi-Fi、蓝牙或任何无线连接
- 不使用 TCP/HTTP 网络协议
- 不使用 adb forward 端口转发
- 手机端 Companion App 不申请 INTERNET 权限
- 手机端是完全被动的，只响应桌面端通过 ADB 发来的命令

## 常见问题

### 手机连接后 DroidLink 没有检测到

1. 确认 USB 线是数据线（非纯充电线）
2. 确认手机已开启 USB 调试
3. 检查手机上是否弹出了"允许 USB 调试"对话框
4. 尝试拔插 USB 线
5. 在手机上将 USB 模式切换为"文件传输"或"MTP"

### Companion App 安装失败

- **提示"通过 USB 安装"被禁用**：在手机设置 > 开发者选项中，开启"通过 USB 安装应用"
- **提示安装验证失败**：在手机设置 > 安全中，临时关闭"验证应用安装"

### 屏幕镜像卡顿

- 降低分辨率设置（如 1280 或 720）
- 降低比特率（如 4 Mbps）
- 确保 USB 线支持 USB 2.0 或更高速度

### 剪贴板不工作

- Android 10+ 对后台剪贴板访问有限制，确保 Companion App 的服务正在运行
- 在手机上打开 Companion App，确认服务状态为"运行中"

### 中文输入不工作

1. 确认已在手机设置中启用 DroidLinkIME
2. 确认当前输入焦点在手机的输入框中
3. 确认已切换到 DroidLink 输入法（非手机原生输入法）

### 联系人/短信/通话记录为空

- 检查 Companion App 是否已授予对应权限
- 部分国产手机有额外的权限管理系统，需要在手机管家/安全中心中单独授权
- 查看 Companion App 界面中的权限状态

### 同步数据不更新

- 确认 Companion App 的服务正在运行（通知栏应显示前台服务通知）
- 在仪表盘页面点击"同步"按钮手动触发同步
- 建议将 Companion App 加入电池优化白名单，防止系统杀后台

## 卸载

### 桌面端
- Windows：控制面板 > 程序和功能 > DroidLink > 卸载
- macOS：将 DroidLink 从 Applications 拖入废纸篓
- Linux：`sudo dpkg -r droidlink` 或删除 AppImage 文件

### Android 端
- 手机设置 > 应用 > DroidLink Companion > 卸载
- 或在桌面端 DroidLink 断开连接后，在手机上长按应用图标卸载
